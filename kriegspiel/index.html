Kriegspiel

这是一个基于Qt6的联机军棋（翻棋）系统，能够根据IP地址寻找对手进行在线游戏。此系统配备超时计数功能和自动判定功能。
This is an online military chess (flip chess) system based on Qt6, which can find opponents to play online games according to the IP address. This system is equipped with timeout counting function and automatic determination function.
